# MovieApp
## 설명
  TMDB API를 통해 영화정보를 불러오고 마음에 드는 영화정보를 저장할 수 있음.
  저장된 영화를 검색할 수 있음.
  
## 사용기술
  - Retrofit
  - Room
  - MVVM
  - Navigation
